Get started with IOXIO.io

Learn how to share, consume and collaborate with data. Explore data sources, sign up, and get your data moving!

Create an account

Creating an account is free and fast. You can create a new account on the fly if you don't already have one. Please input all the required details and you will be logged in to the service as a new user. Remember the email address you used to sign up to the system, as you will need it when joining a group, or requesting a new group for your organization in the next step.

Create an account

Get a group

At the core of access control on IOXIO.io are groups. Every data source belongs to a group, and groups request access to consume data from data sources controlled by other groups.

Each user can join an organization's existing group or request a new group. For IOXIO.io these groups can be requested via email or through our community Slack . For fastest verification of your group, please send an email to support@ioxio.com and tell us the name of the group you would like to be created for you.

Contact support

Start sharing data

Data sharing is free, and you can share your data as soon as you have a group set up.

Once you have an API that follows one of the existing data definitions on IOXIO.io you can start sharing your data! Check out the available data definitions from the definition viewer . If you can't find a definition that covers your use case, please contact us at support@ioxio.com or through our community Slack and describe your usecase, so we can solve the problem together.

Once you have created the API, you can log in to the Dataspace to publish your data source.

Share data

Start consuming data

To consume data from other groups, you will need to upgrade your subscription. To get a quote, please contact us at sales@ioxio.com .

You can find the available data sources in the discover data sources section. You can check the tecnical contact email from the data source listing. Remember to include in the email your group's name and which data source you want access to. Once you have been granted access, you can use the access control keys section to generate API tokens. We also have guides for consuming data to help you get started.
